指针的形式

我们知道指针的形式可以是 int, char, short...等等之类的，但是其实指针的形式也可以是 void，void pointer 的作用其实很大，当我们认为这个变量可以是其他类型的时候，我们可以先将这个变量设定为void pointer 的类型，然後这个类型可以是int啊或者是 char 啊，我会在接下来的系列来一一讲解 void pointer 好用的地方，这一章我先讲解 void pointer 的一个简单的形式。

假设我们有一个程序，这个程序声明了一个整数变量 a，我们想要用一个指针变量来指这个变量 a:

我们可以创建一个 void pointer， *void_ptr 来指向这个变量 a ，然后再用 printf 把它印出来：

当然我们实际去跑这个档案的话会得到错误的信息，因为我们需要将它转成 int 的类型

所以我们将void_ptr 改写成 int *void_ptr，记住这是第一步，将void_ptr 改写成 int pointer 的类型，但是这只是将指针改成整数类型而已，如果我们要指针的数值，记得还要再加上一个星号*，也就是*(int *)void_ptr ，这样才能将指针的数值抽取出来。
